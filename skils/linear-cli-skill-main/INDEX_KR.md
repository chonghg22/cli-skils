# INDEX_KR

## 한 줄 설명
Linear를 가볍고 단순한 CLI 방식으로 다루기 위한 스킬 모음입니다.

## 언제 쓰면 되나
- Linear 이슈를 빠르게 조회, 생성, 수정, 삭제하고 싶을 때
- 복잡한 워크플로보다 CLI 중심의 단순 조작이 필요할 때
- 팀, 프로젝트, 이슈 목록을 바로 터미널에서 보고 싶을 때

## 이런 상황에 특히 적합
- "이슈 몇 개만 빨리 만들자"
- "현재 진행 중인 이슈 상태만 확인하자"
- 스크립트나 자동화에서 가벼운 Linear 조작이 필요할 때

## 이럴 때는 다른 폴더가 더 맞음
- 프로젝트 관리 규칙, 라벨 체계, 설명 품질까지 챙기려면 `linear-claude-skill-main`
- 작업 계획 실행과 병렬 개발은 `subagent-driven-development`

## 빠른 예시
- 특정 팀의 최근 이슈 목록 조회
- 이슈 1건 생성 후 담당자/상태 지정
- 프로젝트 목록 확인

## 메모
- 이 폴더 안에는 실제 CLI 구현과 문서가 함께 들어 있습니다.
- 가볍고 빠른 작업에는 이쪽이 더 적합합니다.
