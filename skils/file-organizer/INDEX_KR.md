# INDEX_KR

## 한 줄 설명
컴퓨터 안의 파일과 폴더를 더 찾기 쉽게 정리하고, 중복 파일과 어수선한 구조를 정리하는 스킬입니다.

## 언제 쓰면 되나
- `Downloads`나 `Documents`가 너무 지저분할 때
- 파일이 여기저기 흩어져 있어서 찾기 어려울 때
- 같은 파일이 중복으로 많이 쌓였을 때
- 프로젝트 폴더 구조를 새로 정리하고 싶을 때

## 이런 상황에 특히 적합
- 오래 쌓인 다운로드 폴더 정리
- 문서, 이미지, 압축파일을 유형별로 분류
- 보관할 파일과 버릴 파일을 나누기 전 분석

## 이럴 때는 다른 폴더가 더 맞음
- Git 저장소 안의 개발 워크플로 분리는 `using-git-worktrees`
- Git 변경 이력 요약은 `changelog-generator`

## 빠른 예시
- "Downloads 폴더 정리해줘"
- "중복 파일 찾아서 어떤 걸 남길지 추천해줘"
- "프로젝트 아카이브용 폴더 구조 제안해줘"

## 메모
- 삭제는 항상 신중해야 하므로, 실제 삭제 전 확인 절차가 중요합니다.
